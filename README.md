:shopping_cart: STUK Front — STUK E-commerce Microservices

Frontend web do projeto STUK, desenvolvido em React. Responsável por toda a interface do usuário, consumindo os microserviços da arquitetura STUK via API.

Este serviço gerencia:

Fluxo de autenticação (Login / Cadastro)
Gestão de usuarios dentro da plataforma

----------------------------------------------------------------------------------------------------------------------------

:computer: Tecnologias utilizadas
React
JavaScript (ES6+)
CSS3
HTML5
React Router DOM
Axios (consumo de APIs REST)
Context API / Hooks

----------------------------------------------------------------------------------------------------------------------------

:clipboard: Funcionalidades

:house: Página Inicial
Exibição dos produtos disponíveis na plataforma com listagem dinâmica consumida via API.

:unlock: Autenticação
Integração com o microserviço de autenticação STUK para login e cadastro de usuários, com gerenciamento de token JWT no frontend.
